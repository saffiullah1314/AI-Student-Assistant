# 🎓 AI Student Personal Assistant

An AI-powered assistant that reads WhatsApp & Gmail messages,
detects important academic & personal information, and sends
daily summaries in Urdu Roman.

## Features
- 📧 Gmail integration
- 💬 WhatsApp integration  
- 🤖 AI detection (Gemini)
- 📋 Auto task creation
- 📅 Daily study plan
- 🌐 Web dashboard

## Tech Stack
- n8n (Automation)
- Gemini API (AI)
- MongoDB Atlas (Database)
- React + Vercel (Dashboard)
- Railway (Hosting)
